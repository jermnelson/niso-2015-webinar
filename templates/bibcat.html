In the fall of 2014, the Library of Congress solicited bids for a new BIBFRAME Search and Display system. Aaron Schmidt of Influx Library User Interface Design and Jeremy Nelson successful proposal was the basis for open-source BIBFRAME Catalog (shortened to BIBCAT) and a backend BIBFRAME Datastore that has a Semantic Server REST API to communicate using Fedora 4 as a linked-data platform with a Elasticsearch and Apache Fuseki SPARQL endpoint.

BIBCAT

BIBCAT, the BIBFRAME Catalog, is a Flask application that dynamically generates a HTML5 web application using Bootstrap, and Knockout.js.
