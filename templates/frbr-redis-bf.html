Almost immediately, development work began on first automating the MARC record manipulation for loading into CC's legacy ILS as well as work on a thesis submission web application.

FRBR

FRBR, short for Functional Requirements for Bibliographic Records, is a resource-entity representation for bibliographic entities that is used in Resource Description and Access.



Redis is an open-source key-value NoSQL data-storage technology where unique keys are associated can be associated with different data structure values like strings, lists, sets, hashes, that enable access to data because Redis stores everything in a computer's memory.

BIBFRAME

BIBFRAME

BIBFRAME, short Bibliographic Framework, is a Library of Congress initiative to replace MARC 21 with a Linked Data vocabulary. BIBFRAME is made up four major classes, Works, Instances, Annotations, and Authorities.
